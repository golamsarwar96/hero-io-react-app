# SOFT.IO


## Soft.Io is a sleek, app-focused website that lets users explore, browse, and install a wide range of applications across different categories. Whether you're looking for productivity tools, utilities, entertainment apps, or developer software, Soft.Io brings everything together in a clean and intuitive interface — making app discovery fast, simple, and enjoyable.

# Technology 

1. JavaScript
2. React JS
3. React Router
4. React Toastify
5. Tailwind
